# Chinese Text with Pinyin

This page demonstrates the zh-cn code block feature with automatic pinyin annotations and tone colors.

## Basic Example

```zh-cn
你好世界
```

## Longer Text

```zh-cn
学习中文很有意思。每天练习一点点，慢慢就会进步。
```

## Tones Example

```zh-cn
妈麻马骂吗
```

The five tones are color-coded:
- 1st tone (high level): Phoenix flame
- 2nd tone (rising): Sea foam
- 3rd tone (falling-rising): Deep amethyst
- 4th tone (falling): Soft grey
- Neutral tone: Bright citrine

## Settings

Use the toggle button in the bottom-right corner to:
- Change pinyin visibility (always / hover / hidden)
- Turn tone colors on or off

Your preferences are saved and persist across visits.
